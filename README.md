# Qudit Library v0.1.0

This library extends Google's Cirq framework to support **qudits**.

You can check the documentation at [https://qudite.pages.dev/].

---

## Installation

See the `DOCUMENTATION.md`.

Clone this repository and run:

```bash
pip install qudit
```
